# 开发启动说明

## 1. 准备远程 MySQL

先在远程 MySQL 中创建数据库：

```sql
CREATE DATABASE activation_system CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

然后复制 `部署配置/.env.example` 为 `.env` 或直接修改 compose 引用的环境变量。

## 2. Docker 热更新启动

```powershell
docker compose -f docker-compose.dev.yml up --build
```

- 管理后台：http://localhost:5173
- 后端接口：http://localhost:3000/api

开发模式已经开启 polling，Windows 下修改代码后页面和后端都会自动更新。

管理后台的 Vite host 和端口已经写在 `管理后台/package.json` 的 `dev` 脚本里：

```json
"dev": "vite --host 0.0.0.0 --port 5173"
```

## 3. 本机不使用 Docker 启动

PowerShell 若拦截 `npm.ps1`，请使用 `npm.cmd`：

```powershell
npm.cmd install
npm.cmd run dev -w 后端服务
npm.cmd run dev -w 管理后台
```
