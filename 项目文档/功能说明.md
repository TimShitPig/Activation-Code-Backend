# 功能说明

## 管理后台

- 系统初始化：第一次部署后在网页填写 MySQL、管理员账号、机器人密钥。
- 首页概览：查看激活码总数、未使用、已激活、部分使用、禁用和今日激活。
- 激活码列表：支持搜索、状态筛选、分页、禁用、启用、删除、批量操作、导出。
- 生成激活码：支持日卡、三天卡、周卡、月卡、季卡、半年卡、年卡。
- 日志中心：展示管理员操作日志、机器人激活日志、机器人校验日志。
- 管理员设置：查看当前管理员，修改密码。

## Docker 热更新

`docker-compose.dev.yml` 使用源码挂载：

- `后端服务` 修改后，NestJS watch 自动重启。
- `管理后台` 修改后，Vite HMR 自动刷新页面。

Windows 下已设置：

```yaml
CHOKIDAR_USEPOLLING: "true"
WATCHPACK_POLLING: "true"
```

这样 Docker Desktop 文件监听不稳定时也能更新。
