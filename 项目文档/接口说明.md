# 接口说明

所有接口默认返回中文提示：

```json
{
  "success": true,
  "message": "操作成功",
  "data": {}
}
```

## 管理员接口

## 初始化接口

- `GET /api/setup/status` 查看系统是否已初始化
- `POST /api/setup/test-database` 测试 MySQL 连接
- `POST /api/setup/initialize` 保存网页初始化配置并创建数据表

- `POST /api/admin/auth/login` 管理员登录
- `GET /api/admin/profile` 获取当前管理员
- `POST /api/admin/change-password` 修改密码

## 激活码管理接口

- `POST /api/admin/codes/generate` 生成激活码
- `GET /api/admin/codes` 查询激活码列表
- `GET /api/admin/codes/stats` 首页统计
- `PATCH /api/admin/codes/:id/disable` 禁用激活码
- `PATCH /api/admin/codes/:id/enable` 启用激活码
- `DELETE /api/admin/codes/:id` 删除激活码
- `POST /api/admin/codes/bulk` 批量操作
- `GET /api/admin/codes/export` 导出 CSV

## 机器人接口

机器人接口需要请求头：

```http
X-API-Token: 部署配置中的 ROBOT_API_TOKEN
```

- `POST /api/client/activate` QQ 号激活
- `POST /api/client/check` QQ 号校验

### QQ 号激活示例

```http
POST /api/client/activate
X-API-Token: please_change_robot_token
Content-Type: application/json

{
  "subjectType": "qq",
  "subjectId": "123456789",
  "code": "ABCD-EFGH-IJKL-MNOP-QRSTUVWX"
}
```

成功返回：

```json
{
  "success": true,
  "message": "操作成功",
  "data": {
    "active": true,
    "subjectType": "qq",
    "subjectId": "123456789",
    "expiresAt": "2026-06-15T08:00:00.000Z",
    "message": "激活成功，会员时长已叠加"
  }
}
```

### QQ 号校验示例

```http
POST /api/client/check
X-API-Token: please_change_robot_token
Content-Type: application/json

{
  "subjectType": "qq",
  "subjectId": "123456789"
}
```

成功返回：

```json
{
  "success": true,
  "message": "操作成功",
  "data": {
    "active": true,
    "subjectType": "qq",
    "subjectId": "123456789",
    "expiresAt": "2026-06-15T08:00:00.000Z",
    "message": "会员有效"
  }
}
```
